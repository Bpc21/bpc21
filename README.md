## About
I’m a **full-stack engineer** who builds production web apps with a heavy bias toward **cloud reliability**, **automation**, and **security-minded** engineering.

**What I’m good at (the useful stuff):**
- Shipping **Next.js 14** apps (App Router) with clean UI and sane state management.
- Designing cloud backends on **AWS Amplify** (Cognito, AppSync, DynamoDB, Lambda, S3, EventBridge).
- Integrating billing and membership systems with **Stripe** (webhooks, entitlements, access control).
- Building automation where humans are the bottleneck (n8n / Zapier / Make / Power Automate).
- Running real-world “ops” work: servers, networking, monitoring, incident-proofing, and hardening.

**Current focus:** cloud + security (because broken systems don’t care how pretty your UI is).
